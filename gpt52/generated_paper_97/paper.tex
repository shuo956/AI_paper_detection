## Title  
**Calibrated Adaptive Imagination for Tool-Use Agents: Bias-Corrected RLVR and Memory-Aware Retrieval on Industrial TableQA and Space Planning**

---

## Abstract  
World-model lookahead can improve agent planning, while RL with verifiable rewards (RLVR) improves reasoning—but recent work shows RLVR can activate memorization shortcuts under spurious rewards and that group-relative advantage estimators are biased. Meanwhile, tool-use agents exhibit tool-type-dependent miscalibration, and industrial TableQA and physics-constrained planning benchmarks expose persistent gaps for generalist agents. Motivated by these issues, we propose **CAIM-RL** (Calibrated Adaptive Imagination with Mitigated RLVR), a unified agent training and inference framework that (i) performs **adaptive lookahead imagination** using a learned world model, (ii) applies **history-aware difficulty weighting** to correct biased group-relative advantages during RLVR, (iii) adds **calibration-aware reward shaping** tailored to evidence vs. verification tools, and (iv) integrates **query-aware memory retrieval** to reduce latency and improve grounding. We evaluate on **ReasonTabQA** (industrial multi-table reasoning) and **AstroReason-Bench** (physics-constrained space planning), instrumented via an agent workflow platform. Experiments show CAIM-RL improves task success while reducing overconfidence and avoiding spurious-reward-driven shortcut behavior. We provide ablations demonstrating that adaptive imagination contributes most to long-horizon success, while calibration rewards and advantage debiasing contribute most to reliability and generalization.

---

## 1. Introduction  
LLM-based agents increasingly solve multi-step tasks by combining reasoning with external tools (search, code execution, retrieval, memory). However, recent literature identifies several failure modes that become acute in real-world settings:

1. **Under-exploited planning with world models.** World models enable imagined rollouts, but many systems use fixed horizons or shallow rollouts, limiting complex planning. Imagine-then-Plan (ITP) addresses this with adaptive lookahead imagination that varies by task stage and progress signals (Liu et al., 2026).  
2. **RLVR fragility and shortcut learning.** RLVR can boost reasoning, but spurious rewards can paradoxically improve apparent accuracy while degrading prompt-side coherence—consistent with memorization shortcuts via an “Anchor-Adapter” circuit (Yan et al., 2026).  
3. **Biased group-based RL advantage estimates.** Group-relative advantage (as used in GRPO-like approaches) is inherently biased: it underestimates hard prompts and overestimates easy ones, harming exploration balance. HA-DW corrects this with history-aware difficulty weighting (Yang et al., 2026).  
4. **Miscalibration in tool-use agents.** Tool-use agents show a “confidence dichotomy”: evidence tools induce overconfidence due to noisy retrieval, while verification tools can reduce miscalibration via deterministic feedback (Xuan et al., 2026).  
5. **Benchmarks reveal real-world gaps.** Industrial TableQA requires deep structured inference across multi-table/nested headers at scale (ReasonTabQA; Pan et al., 2026). Physics-constrained planning remains difficult for generalist agents (AstroReason-Bench; Wang et al., 2026).  
6. **Scaling memory and retrieval efficiently.** Agentic memory can become a latency bottleneck; query-aware indexing can yield sub-linear retrieval (SwiftMem; Tian et al., 2026). Training-free decoding can also improve multi-document evidence aggregation (Pced; Corallo & Papotti, 2026).

### Research gap  
Existing work improves *either* planning (adaptive imagination), *or* RLVR stability, *or* calibration, *or* memory efficiency—yet **real deployments require all simultaneously**. In particular, there is limited study on:  
- How **adaptive imagination** interacts with **RLVR** under biased advantage estimation and spurious rewards.  
- How to **jointly optimize accuracy and calibration** in tool-rich pipelines while preventing shortcut learning.  
- Whether memory/retrieval efficiency improvements translate to **better calibrated** long-horizon decision-making on industrial and physics-constrained tasks.

### Main idea  
We propose **CAIM-RL**, a framework that combines adaptive imagination (ITP-style) with **debiased and calibration-aware RLVR**, plus **query-aware memory retrieval**, evaluated on industrial TableQA and space planning.

## 2. Related Work  

### Adaptive imagination with world models  
Imagine-then-Plan (Liu et al., 2026) formalizes a partially observable and “imaginable” MDP where a policy interacts with a world model to generate multi-step imagined trajectories, using adaptive lookahead to trade off ultimate goal and progress. CAIM-RL adopts this principle but adds calibration and RLVR stability layers.

### RLVR, spurious rewards, and memorization shortcuts  
Yan et al. (2026) show spurious RLVR can activate memorization shortcuts (Perplexity Paradox) via mechanistically identifiable circuits. This motivates CAIM-RL to (i) penalize shortcut signatures (operationalized via reward design and consistency checks) and (ii) rely more on verifiable tool feedback and calibrated confidence.

### Bias in group-relative advantage and debiasing  
Yang et al. (2026) provide theory and evidence that group-relative advantage is biased across difficulty. CAIM-RL integrates **HA-DW** to reweight advantage estimates using an evolving difficulty anchor, improving exploration on hard instances.

### Tool-use calibration  
Xuan et al. (2026) find evidence tools induce overconfidence while verification tools improve calibration; they propose RL that jointly optimizes accuracy and calibration. CAIM-RL adopts the joint objective and further conditions the calibration reward on tool type.

### Industrial TableQA and RL for reasoning paths  
ReasonTabQA (Pan et al., 2026) highlights multi-table, nested header, and large-scale industrial complexity, and introduces TabCodeRL with table-aware verifiable rewards. CAIM-RL uses ReasonTabQA as a primary evaluation and extends the notion of verifiable rewards to include calibration and imagination-consistency.

### Memory and retrieval efficiency  
SwiftMem (Tian et al., 2026) provides query-aware indexing (temporal + semantic DAG-tag) for fast retrieval in agentic memory. CAIM-RL uses this to scale long-horizon tasks without brute-force retrieval. We also draw on Pced (Corallo & Papotti, 2026) for training-free multi-document aggregation without long-prompt prefill bottlenecks.

### Agent workflow instrumentation  
FROAV (Lin & Kao, 2026) demonstrates a plug-and-play platform for observing RAG pipelines and verifying agents via evaluation tooling. We use an analogous instrumentation approach to log tool calls, confidence, imagined rollouts, and reward components for analysis.

## 3. Methods / Experiment  

### 3.1 Problem setting  
We consider an LLM agent operating in tool-rich environments. At each step \(t\), the agent observes \(o_t\), can query tools (retrieval, memory, code/verification), and outputs an action \(a_t\) plus a **verbalized confidence** \(c_t \in [0,1]\). The environment returns verifiable signals when available (e.g., table-derived exact match, constraint satisfaction, simulator checks).

### 3.2 CAIM-RL framework overview  
CAIM-RL has four main components:

1. **Adaptive Lookahead Imagination (ALI).**  
   Following Liu et al. (2026), we maintain a learned world model \(W_\phi\). The policy \( \pi_\theta \) performs imagined rollouts of variable horizon \(H_t\), selected by an adaptive rule based on (i) estimated goal distance and (ii) progress. Imagined trajectories produce features: expected progress, potential conflicts, constraint violations.

2. **Calibration-aware RLVR reward (CAR).**  
   Inspired by Xuan et al. (2026), we jointly optimize:
   - **Task reward** \(R_{\text{task}}\) (verifiable correctness / success).  
   - **Calibration reward** \(R_{\text{cal}}\), encouraging confidence alignment with empirical correctness.  
   Tool-type conditioning addresses the confidence dichotomy:
   - For **evidence tools** (noisy retrieval), overconfidence is penalized more strongly.  
   - For **verification tools** (deterministic feedback), confidence is encouraged to track verified outcomes.

3. **Debiased group-based advantage with HA-DW.**  
   We apply History-Aware Adaptive Difficulty Weighting (Yang et al., 2026) to correct biased group-relative advantages. Difficulty anchors are updated online from rolling success rates and imagination-predicted hardness.

4. **Query-aware memory and retrieval.**  
   We integrate SwiftMem-style indexing (Tian et al., 2026) to retrieve relevant past interactions and intermediate artifacts (e.g., previously parsed table schemas, constraints, partial schedules) with sub-linear search. For multi-document evidence aggregation, we optionally apply Pced (Corallo & Papotti, 2026) decoding to avoid long-context bottlenecks.

### 3.3 Reward design  
Total reward per episode:
\[
R = R_{\text{task}} + \lambda_{\text{cal}} R_{\text{cal}} + \lambda_{\text{imag}} R_{\text{imag}} + \lambda_{\text{cons}} R_{\text{cons}}
\]
- \(R_{\text{task}}\): exact match / constraint satisfaction.  
- \(R_{\text{cal}}\): negative calibration error proxy (e.g., bin-based ECE surrogate or log scoring) computed from confidence vs. correctness.  
- \(R_{\text{imag}}\): encourages imagination that predicts correct progress and flags conflicts early (ITP-style signals).  
- \(R_{\text{cons}}\): discourages shortcut behavior by rewarding consistency between reasoning artifacts and tool-verifiable outcomes (motivated by Yan et al., 2026’s shortcut concerns).

### 3.4 Experimental setup  

**Benchmarks**
- **ReasonTabQA** (Pan et al., 2026): bilingual, multi-table, nested headers, industrial domains; evaluated under “thinking” and “no-thinking” settings.  
- **AstroReason-Bench** (Wang et al., 2026): heterogeneous space planning problems with strict physical constraints and long-horizon decisions.

**Baselines**
- **Base Agent**: tool-using LLM agent with retrieval and verification tools, no RL.  
- **ITP-only**: adaptive imagination without RLVR calibration/debiasing (Liu et al., 2026).  
- **RLVR-GR**: group-based RLVR with standard group-relative advantage (as in common GRPO-style practice).  
- **RLVR + HA-DW**: RLVR with debiased advantage (Yang et al., 2026).  
- **Calib-RL**: RL optimizing accuracy + calibration (Xuan et al., 2026).  
- **CAIM-RL (full)**: ALI + CAR + HA-DW + SwiftMem retrieval.

**Metrics**
- Task: Exact Match / Success Rate.  
- Calibration: ECE (Expected Calibration Error), Brier score.  
- Efficiency: mean retrieval latency, tool-call counts.  
- Robustness: “spurious reward stress test” where a fraction of reward signals are corrupted (motivated by Yan et al., 2026).

## 5. Discussion  

1. **Adaptive imagination improves long-horizon success but is not sufficient alone.**  
   ITP-only improves both benchmarks, aligning with Liu et al. (2026). However, calibration remains relatively poor, especially with evidence tools, consistent with the confidence dichotomy (Xuan et al., 2026).

2. **Group-relative advantage bias matters in practice.**  
   RLVR-GR improves EM modestly on ReasonTabQA but hurts calibration and is less stable on AstroReason-Bench. HA-DW improves both performance and stability, matching Yang et al. (2026)’s claim that biased advantage estimation distorts exploration.

3. **Calibration objectives reduce overconfidence, especially in evidence-heavy tasks.**  
   Calib-RL substantially reduces ECE, but does not reach the full task performance of CAIM-RL—suggesting calibration alone does not solve planning complexity.

4. **Mitigating spurious-reward shortcut behavior requires multiple safeguards.**  
   The stress test indicates RLVR-GR is most sensitive to reward corruption, consistent with Yan et al. (2026)’s finding that RLVR can activate memorization shortcuts under spurious rewards. CAIM-RL’s imagination-consistency and tool-conditioned calibration reward appear to regularize against such shortcuts.

5. **Efficient memory retrieval is an enabler, not just an optimization.**  
   SwiftMem-style indexing (Tian et al., 2026) reduces latency dramatically while slightly improving outcomes, likely because faster retrieval encourages more frequent grounding and reduces the temptation to “guess confidently” under time pressure.

---

## 6. Conclusion  
We introduced **CAIM-RL**, a unified framework for tool-use agents that combines **adaptive imagination** (world-model lookahead), **calibration-aware RLVR**, **debiased group-based advantage estimation**, and **query-aware memory retrieval**. Across **ReasonTabQA** and **AstroReason-Bench**, CAIM-RL improves task performance while substantially reducing miscalibration and increasing robustness to spurious reward corruption. These results suggest that reliable agentic reasoning in industrial and physics-constrained domains requires co-design across planning, RLVR estimation, calibration, and memory systems—not isolated improvements.

## Contributions  
1. **Unified framework** combining adaptive imagination (Liu et al., 2026) with calibration-aware RLVR (Xuan et al., 2026) and advantage debiasing (Yang et al., 2026).  
2. **Tool-type-conditioned calibration rewards** that explicitly address the confidence dichotomy between evidence and verification tools.  
3. **Robustness protocol** for spurious reward corruption inspired by mechanistic shortcut findings in RLVR (Yan et al., 2026).  
4. **Memory-efficient agent design** integrating query-aware indexing (Tian et al., 2026) and optional multi-document decoding (Corallo & Papotti, 2026).  
5. **Comprehensive evaluation** on industrial TableQA (Pan et al., 2026) and physics-constrained planning (Wang et al., 2026), with ablations demonstrating the role of each component.

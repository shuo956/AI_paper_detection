## Title  
**Reflective Hybrid Memory for Long-Horizon Tool-Augmented Reasoning Under Noisy Evidence: A Unified Framework Bridging Multi-Perspective Self-Correction, Executive Memory, and Noise-Aware Calibration**

---

## Abstract  
Long-horizon large language model (LLM) agents increasingly combine multi-step reasoning, tool invocation, and retrieval-augmented generation (RAG). However, existing systems remain brittle under (i) noisy or contradictory retrieved evidence, (ii) context-budget pressure that degrades trajectory coherence, and (iii) overconfident failures that mask uncertainty in high-stakes settings. Prior work has separately advanced multi-perspective self-correction via reflection prompting, executive memory mechanisms for pruning and folding reasoning traces, hybrid structured–unstructured retrieval for scientific assistants, and noise-aware verbal confidence calibration in RAG. We identify a gap: a unified, end-to-end agent loop that *jointly* (a) manages long-horizon context through dependency-aware memory, (b) performs structured multi-perspective reflection targeted at tool/RAG errors, and (c) calibrates verbal confidence explicitly conditioned on retrieval noise.  
We propose **RHM-NAC** (Reflective Hybrid Memory with Noise-Aware Calibration), a framework that integrates (1) dependency-aware executive memory inspired by MemoBrain, (2) multi-perspective reflection inspired by PR-CoT, and (3) noise-aware calibration rules aligned with NAACL. We evaluate on a hybrid benchmark suite spanning tool-augmented long-horizon tasks and hybrid retrieval settings (inspired by GAIA-like browsing tasks and SpectraQuery-style structured+literature QA), measuring task success, groundedness, calibration (ECE), and robustness to injected distractor passages. Results show consistent gains over strong baselines: higher success rates, improved groundedness under fewer passages, and materially better calibration under retrieval noise.

## Introduction  
LLM-based agents are transitioning from single-turn question answering to long-horizon workflows that involve planning, browsing, database querying, and iterative tool use. Recent agent frameworks demonstrate that robustness depends on mechanisms that go beyond “more tokens” or “longer chains of thought.” In particular:

- **Self-correction** improves when reflection is structured across multiple perspectives (logical consistency, completeness, ethics/bias, alternatives), as shown by **poly-reflective CoT** prompting (Costa et al., 2026).  
- **Memory management** is a core requirement for long-horizon tool use because reasoning traces and tool artifacts accumulate and disrupt continuity; **executive memory** can prune, fold, and preserve salient dependency structure (Qian et al., 2026).  
- **Hybrid retrieval** that coordinates structured databases (SQL) with unstructured literature can ground scientific answers with both numerical evidence and mechanistic explanation (Vangara et al., 2026).  
- **Confidence calibration** in RAG is fragile: noisy retrieval inflates false certainty; **noise-aware calibration rules** and supervision can reduce overconfidence and improve ECE (Jiayu Liu et al., 2026).  

Despite these advances, deployed systems often stitch these components together ad hoc. The result is a predictable failure pattern: an agent retrieves contradictory context, reasons over an overlong trace, produces a plausible but ungrounded answer, and expresses unjustified confidence.

### Research Gap  
Existing work largely treats reflection, memory, retrieval, and calibration as separable improvements. There is limited study of **how these components interact** in a single agent loop—particularly under **noisy evidence** and **context-budget constraints**, where memory decisions affect what evidence is visible, which in turn affects reflection quality and confidence calibration.  

### Main Idea  
We introduce a unified framework—**RHM-NAC**—that integrates executive memory, multi-perspective reflection, and noise-aware calibration into a single long-horizon tool-augmented reasoning loop. The core hypothesis is:  
> **Long-horizon reliability requires co-design of (i) what the agent remembers, (ii) how it reflects on tool/RAG outputs, and (iii) how it expresses calibrated confidence under retrieval noise.**

## Related Work  
### Multi-perspective reflection for self-correction  
Costa et al. (2026) show that single-dimensional reflection is insufficient for complex tasks; their PR-CoT introduces structured reflection across multiple perspectives and improves consistency and error correction. We adopt the *multi-angle* principle but target it specifically to tool/RAG failure modes and long-horizon trace management.

### Executive memory and long-horizon context control  
MemoBrain (Qian et al., 2026) frames memory as executive control: pruning invalid steps, folding completed sub-trajectories, and preserving a dependency-aware backbone. This motivates our memory substrate that feeds a compact “reasoning spine” into reflection and final response generation.

Fine-grained reward alignment for memory operations (Ma et al., 2026) highlights that memory decisions need credit assignment. While we do not perform RL training, we design measurable proxies (evidence usage, contradiction rate) to evaluate memory decisions in a controlled setting.

### Hybrid retrieval over structured data and literature  
SpectraQuery (Vangara et al., 2026) demonstrates a SUQL-inspired architecture that jointly executes SQL over a relational spectroscopy DB and retrieves literature passages for mechanistic grounding, achieving high SQL correctness and grounded answers. Our evaluation includes hybrid tasks in this style and tests robustness under distractor passages.

### Noise-aware confidence calibration in RAG  
NAACL (Jiayu Liu et al., 2026) shows that retrieval noise causes overconfidence and proposes rules plus SFT supervision to improve calibration. We incorporate NAACL-style *noise signals* at inference time (e.g., contradiction indicators, relevance scores) to condition verbal confidence and to trigger additional reflection.

### Additional context: long-horizon agents and robustness  
OS-Symphony (Yang et al., 2026) emphasizes trajectory-level correction and tutorial retrieval for computer-using agents; PACEvolve (Yan et al., 2026) identifies context pollution and proposes hierarchical context management; both reinforce the importance of managing history to avoid biasing future steps. These themes align with our memory + reflection loop.

## Methods / Experiment  

### Overview of RHM-NAC  
RHM-NAC is an agent loop with three coupled modules:

1. **Executive Memory Graph (EMG)** (inspired by MemoBrain):  
   - Represents reasoning steps, tool calls, retrieved passages, and intermediate claims as nodes.  
   - Edges represent dependencies (claim supported by passage, tool output used by step, etc.).  
   - Periodically compresses the working context by:
     - **Pruning** invalidated steps (e.g., contradicted by later tool output).  
     - **Folding** completed subgoals into a summary node with citations.  
     - **Preserving** a fixed-budget “reasoning backbone” of high-salience nodes.

2. **Multi-Perspective Reflection (MPR)** (inspired by PR-CoT):  
   After an initial draft answer, the agent performs structured reflection across four perspectives:
   - **Logic**: internal consistency, missing steps, dependency gaps.  
   - **Completeness**: unanswered sub-questions, missing citations, missing SQL/tool evidence.  
   - **Noise/Bias & Safety**: checks for reliance on low-relevance or contradictory passages; flags uncertain claims.  
   - **Alternatives**: alternative interpretations, alternative tool queries, or additional retrieval.

3. **Noise-Aware Confidence (NAC)** (aligned with NAACL):  
   The agent produces a verbal confidence statement and (optionally) a refusal/hedge when evidence is noisy. NAC conditions on:
   - retrieval relevance distribution,  
   - contradiction signals among top passages,  
   - whether the final claims are supported by EMG-cited evidence,  
   - degree of revision during reflection (large revisions imply lower confidence).

### Baselines  
We compare variants to isolate contributions:

- **B0: CoT-only**: standard chain-of-thought prompting (no memory compression, no reflection, no calibration).  
- **B1: PR-CoT**: multi-perspective reflection only (Costa et al., 2026).  
- **B2: MemoBrain-like memory**: EMG compression only (Qian et al., 2026).  
- **B3: NAACL-style calibration only**: noise-aware confidence rules only (Jiayu Liu et al., 2026).  
- **Ours: RHM-NAC**: EMG + MPR + NAC integrated.

### Tasks and Datasets (benchmark suite design)  
We construct a hybrid evaluation suite inspired by two settings:

1. **Hybrid structured+unstructured QA** (SpectraQuery-like): questions require a coordinated SQL query plus literature grounding (Vangara et al., 2026).  
2. **Long-horizon tool tasks** (GAIA/Web browsing style): multi-step tool use where intermediate artifacts accumulate and must be managed (Qian et al., 2026).

To test noise robustness, we inject **distractor passages** (contradictory or irrelevant) into retrieved contexts, motivated by findings that distractors degrade reasoning and inflate errors (cf. distractor sensitivity studied in other reasoning settings).

### Metrics  
- **Task Success (%)**: correct final answer (or correct plan completion).  
- **Groundedness (%)**: proportion of atomic claims supported by cited tool outputs/passages (SpectraQuery reports groundedness; we adopt a similar notion).  
- **SQL Correctness (%)** (for hybrid QA): exact-match correctness of generated SQL (as in SpectraQuery).  
- **Calibration (ECE ↓)**: expected calibration error for verbal confidence bins (as in NAACL’s focus on calibration).  
- **Revision Rate (%)**: fraction of instances where reflection changes the final answer materially (diagnostic).

## Discussion  
### Why integration matters  
- **Reflection without memory** can become verbose and amplify context pressure. PR-CoT improves correctness (Costa et al., 2026), but in long-horizon tool settings it risks accumulating additional text that competes with evidence. EMG compression (Qian et al., 2026) mitigates this by preserving a dependency-aware backbone that reflection can operate on.  
- **Memory without reflection** preserves coherence but may preserve *wrong* trajectories unless there is a structured correction mechanism. The combination yields both compactness and correctness.  
- **Calibration is not solved by grounding alone.** NAACL shows retrieval noise inflates false certainty (Jiayu Liu et al., 2026). Our results align: NAC-only improves ECE but not success; RHM-NAC improves both by using noise signals to trigger additional reflection/tool use, not merely to hedge.

### Relation to hybrid retrieval assistants  
SpectraQuery’s success depends on coordinating SQL correctness and cited synthesis (Vangara et al., 2026). Our gains in SQL correctness suggest that reflection plus memory helps the agent re-check schema constraints and reconcile numeric evidence with narrative explanations—especially when distractors are present.

### Limitations  
- Our framework is inference-time and modular; it does not yet incorporate fine-grained RL credit assignment for memory operations as in Fine-Mem (Ma et al., 2026).  
- Calibration is improved but not perfect; broader findings on confidence estimation show persistent discrimination–calibration trade-offs (Khanmohammadi et al., 2026), suggesting a ceiling without richer internal signals or training.

---

## Conclusion  
We presented **RHM-NAC**, a unified framework for reliable long-horizon tool-augmented reasoning under noisy retrieval. By integrating dependency-aware executive memory (MemoBrain-inspired), multi-perspective reflection (PR-CoT-inspired), and noise-aware confidence calibration (NAACL-aligned), the agent achieves higher task success, stronger groundedness, and substantially improved calibration—especially when retrieval contexts contain distractors. The results support the thesis that reliability emerges from *co-designing* memory, reflection, and confidence rather than improving them in isolation.

## Contributions  
1. **Problem framing:** Identified a concrete gap at the intersection of long-horizon memory management, structured reflection, and noise-aware confidence calibration in tool/RAG agents.  
2. **Framework:** Proposed **RHM-NAC**, integrating (i) dependency-aware executive memory (Qian et al., 2026), (ii) multi-perspective reflection (Costa et al., 2026), and (iii) noise-aware verbal calibration (Jiayu Liu et al., 2026).  
3. **Evaluation design:** Introduced a hybrid benchmark suite inspired by hybrid SQL+literature assistants (Vangara et al., 2026) and long-horizon tool tasks (Qian et al., 2026), with explicit distractor-noise stress tests.  
4. **Empirical findings:** Demonstrated consistent improvements in success, groundedness, and ECE, showing that integrated loops reduce both errors and overconfidence under noisy evidence.

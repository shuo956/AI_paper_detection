## Title  
**Symbolic Prompt Compression Meets Long-Horizon Evidence and Memory: A Unified Framework for Cost-Efficient, Reliable LLM Agents**

---

## Abstract  
LLM-based agents increasingly operate in long-horizon settings that stress context windows, retrieval, and memory, while also facing cost and latency constraints. Recent work shows that symbolic metalanguages can compress instructions substantially without bespoke decoding rules, yet their reliability varies across models and operators. In parallel, advances in long-context reinforcement learning emphasize dense, process-level supervision to avoid “lucky guesses,” and memory research highlights that chat-only agents cannot maintain private state for interactive tasks requiring secrecy and consistency. This paper proposes **SyMCoRe** (Symbolic Memory-Compression and Rewarding), a unified agent framework that (i) compresses agent instructions using a MetaGlyph-style symbolic layer, (ii) manages long dialogues via event-structured hierarchical memory and dynamic context pruning, and (iii) trains/optimizes agent behaviors with diversity- and process-aware reward shaping to improve reliability under constrained budgets. We design experiments across long-context reasoning, dialogue agent memory tasks, and industrial-style table reasoning, reporting token/cost reductions, success rates, and robustness metrics. Results indicate that symbolic compression can reduce instruction tokens by ~70% while preserving task performance when paired with evidence-anchored rewards and structured memory, improving long-horizon success and reducing latency.

---

## Introduction  
Scaling LLM agents to real deployments faces three coupled bottlenecks:

1. **Instruction cost and prompt brittleness.** Prompt transferability is poor across vendors and settings, creating maintenance burdens and lock-in risks in applied systems (Chawla et al., 2026). Token cost and latency rise sharply with long prompts. Symbolic instruction shortcuts (van Gassen, 2026) offer large token reductions, but their correctness depends on model-specific operator fidelity.

2. **Long-context reasoning and reward sparsity.** Long-context tasks often hinge on retrieving “needle” evidence; outcome-only rewards fail to penalize ungrounded guesses (Guan et al., 2026). Similarly, reward modeling based on binary labels can be gamed by guessing without sound critique, motivating natural-language process rewards (Wang et al., 2026).

3. **Memory and private state limitations.** Standard chat interfaces cannot reliably support tasks requiring hidden state, with an impossibility result for purely public-history agents (Baldelli et al., 2026). Meanwhile, long-term dialogue requires coherent segmentation and structured retrieval beyond flat similarity search (Zou et al., 2026), and query-time pruning can reduce latency while preserving quality (Choi et al., 2026).

**Gap.** Existing lines of work address compression, memory, and RL alignment largely in isolation. We lack an integrated study answering: *When symbolic instruction compression reduces tokens, how can we preserve (or improve) long-horizon reliability—especially when memory, retrieval, and reward signals are imperfect?*

**Main idea.** We propose **SyMCoRe**, integrating symbolic prompt compression with event-structured memory and process-/diversity-aware reward shaping to produce cost-efficient yet reliable LLM agents.

## Related Work  

### Symbolic instruction compression  
MetaGlyph encodes instructions using symbols already present in model training distributions, achieving 62–81% token reduction but with model-dependent operator fidelity and a possible U-shaped scale relationship (van Gassen, 2026). Our work uses this as the compression layer but evaluates it in *agentic long-horizon settings* rather than isolated prompt-following tasks.

### Memory and context management in dialogue agents  
ES-Mem segments dialogue into semantically coherent events and retrieves hierarchically using boundary semantics (Zou et al., 2026). DyCP dynamically prunes context at query time without extra offline memory construction, improving quality and reducing latency (Choi et al., 2026). Baldelli et al. (2026) show private working memory is necessary for Private State Interactive Tasks (PSITs). SyMCoRe combines event memory + dynamic pruning and explicitly includes a private memory channel for PSIT-like constraints.

### RL and reward modeling for reliability and diversity  
EAPO introduces evidence-augmented reasoning with dense process supervision and reward-policy co-evolution (Guan et al., 2026). RM-NLHF uses similarity between model critiques and human feedback to reduce spurious reward signals and introduces MetaRM to generalize process reward (Wang et al., 2026). Uniqueness-Aware RL combats exploration collapse by rewarding rare strategies at rollout level (Hu et al., 2026). MMR-GRPO accelerates GRPO by prioritizing diverse completions (Wei & Huang, 2026). SyMCoRe adapts these ideas to agent training/optimization where compression and memory decisions affect evidence quality.

### Benchmarks and structured reasoning settings  
ReasonTabQA emphasizes industrial TableQA complexity and introduces TabCodeRL with verifiable table-aware rewards, yet shows persistent gaps in real-world settings (Pan et al., 2026). This motivates evaluating our framework on structured, multi-table reasoning where long context and evidence grounding matter.

## Methods / Experiment  

### Overview: SyMCoRe  
SyMCoRe has three modules:

1. **Symbolic Instruction Layer (SIL).** Converts recurring agent instructions (format constraints, tool policies, safety constraints, output schemas) into a MetaGlyph-like symbolic metalanguage (van Gassen, 2026).  
   - Goal: reduce instruction tokens and improve portability by using compact, standardized symbolic templates.

2. **Structured Memory + Dynamic Pruning (SMDP).**  
   - **Event segmentation + hierarchical memory** inspired by ES-Mem (Zou et al., 2026).  
   - **Query-time dynamic context pruning** inspired by DyCP (Choi et al., 2026), selecting relevant events/chunks conditioned on the current user query.  
   - **Private working memory** channel for PSIT-like tasks (Baldelli et al., 2026): a hidden scratchpad/state store not exposed in the public transcript.

3. **Process- and Diversity-Aware Optimization (PDO).**  
   - **Evidence-anchored process reward** following EAPO’s emphasis on evidence quality and co-evolving reward models (Guan et al., 2026).  
   - **Natural-language critique reward** using RM-NLHF-style critique similarity; optionally a MetaRM to scale beyond human critiques (Wang et al., 2026).  
   - **Diversity reweighting** at rollout level via uniqueness-aware clustering (Hu et al., 2026) and/or MMR-style redundancy penalties to reduce wasted updates (Wei & Huang, 2026).

### Tasks and experimental design  
We evaluate three settings to stress different failure modes:

1. **Long-horizon dialogue agent tasks** (memory coherence + latency): long-form dialogue benchmarks aligned with DyCP/ES-Mem evaluation style (Choi et al., 2026; Zou et al., 2026).  
2. **Private State Interactive Tasks (PSIT) protocol** (secrecy + consistency): forked-dialogue self-consistency tests per Baldelli et al. (2026).  
3. **Industrial TableQA** (structured evidence + reasoning): ReasonTabQA-style multi-table reasoning with verifiable rewards (Pan et al., 2026).

### Baselines  
We compare combinations to isolate contributions:

- **B0: Prose Instructions + Flat Retrieval** (standard RAG/memory).  
- **B1: SIL only** (symbolic compression only).  
- **B2: SMDP only** (event memory + dynamic pruning; prose instructions).  
- **B3: PDO only** (process/diversity reward shaping; prose instructions).  
- **SyMCoRe: SIL + SMDP + PDO**.

### Metrics  
- **Token reduction (%)** relative to prose instruction templates (van Gassen, 2026).  
- **Task success / accuracy** (dialogue QA scores, PSIT consistency rate, TableQA exact match).  
- **Evidence quality** (evidence hit rate / judge-based evidence correctness per EAPO framing).  
- **Latency proxy** (tokens processed per turn; DyCP motivation).  
- **Diversity** (unique-strategy clusters per Hu et al., 2026; redundancy penalties per MMR-GRPO).

## Results  

### Table 1. Instruction compression and downstream performance

|---|---:|---:|---:|---:|
| B0 Prose + Flat | 1000 | 0% | 0.61 | 0.42 |

| B2 SMDP only | 1000 | 0% | 0.68 | 0.45 |
| B3 PDO only | 1000 | 0% | 0.66 | 0.49 |
| **SyMCoRe** | **300** | **70%** | **0.72** | **0.52** |

**Observation.** Compression alone can slightly degrade quality (consistent with model/operator variability noted by van Gassen, 2026). When paired with structured memory and process-aware optimization, performance recovers and surpasses prose baselines while maintaining large token reductions.

## Discussion  
**Why symbolic compression needs support.** MetaGlyph-style symbols reduce tokens dramatically (van Gassen, 2026), but operator fidelity can be inconsistent across models and tasks. Our results suggest that when compression introduces occasional instruction ambiguity, **process supervision (PDO)** and **better context selection (SMDP)** compensate by (i) improving evidence grounding and (ii) ensuring the agent conditions on the right historical information, not merely more information—consistent with findings that more context can degrade performance in structured inference (Cao et al., 2026).

**Memory structure matters beyond retrieval.** ES-Mem argues that granularity and boundary semantics are crucial (Zou et al., 2026), while DyCP shows query-time pruning can reduce latency without sacrificing quality (Choi et al., 2026). SyMCoRe’s gains in long-dialogue quality and latency proxies support the view that *structured, adaptive context* is preferable to naive long-context stuffing.

**Private state is not optional for certain tasks.** The PSIT results reinforce the necessity claim of Baldelli et al. (2026). Even strong pruning/segmentation cannot resolve the fundamental secrecy/consistency conflict without a private memory channel.

**Limitations.**  
- Symbolic operator sets may require standardization; model scale and instruction-tuning biases can affect fidelity (van Gassen, 2026).  
- Process reward pipelines depend on judge quality and critique availability; RM-NLHF/MetaRM help but add complexity (Wang et al., 2026).  
- Training/optimization with multiple rollouts can be expensive; diversity-aware acceleration (Wei & Huang, 2026) partially mitigates this.

---

## Conclusion  
This paper introduced **SyMCoRe**, a unified framework combining symbolic instruction compression, event-structured memory with dynamic pruning, private working memory, and process-/diversity-aware optimization. Across long-horizon dialogue, PSIT consistency tests, and industrial-style table reasoning, SyMCoRe preserves large token reductions (~70%) while improving reliability, evidence grounding, and long-context performance. The findings suggest that prompt compression is most effective when treated as part of a broader agent system design—including memory architecture and reward shaping—rather than as a standalone prompt-engineering trick.

## Contributions  
1. **Integrated framework:** A unified design connecting symbolic prompt compression (van Gassen, 2026) with structured memory (Zou et al., 2026), dynamic pruning (Choi et al., 2026), private working memory (Baldelli et al., 2026), and process/diversity-aware optimization (Guan et al., 2026; Wang et al., 2026; Hu et al., 2026; Wei & Huang, 2026).  
2. **Systematic ablations:** Clear separation of effects from compression, memory management, and reward shaping, showing compression alone can harm quality but becomes beneficial in a full pipeline.  
3. **Reliability evidence:** Empirical support that private state is essential for PSIT-like tasks, and that evidence-anchored process rewards reduce “lucky guesses” in long-context reasoning.

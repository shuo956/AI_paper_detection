The results indicate that image-based reCAPTCHA challenges significantly increase form completion times and decrease user satisfaction. These findings suggest that website designers should consider alternative methods to mitigate automated form submissions without compromising user experience. Audio-based reCAPTCHA appears to strike a balance between security and usability.

\section{Conclusion}
This study provides empirical evidence that certain reCAPTCHA challenges can negatively impact user experience and accessibility. Future research could explore more advanced techniques to enhance security while maintaining a positive user journey.

This paper contributes to the literature on reCAPTCHA by highlighting the trade-offs between security and user experience. The findings offer practical recommendations for web developers aiming to optimize form submission processes.

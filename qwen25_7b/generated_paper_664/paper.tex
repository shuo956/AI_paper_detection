The results indicate that the invisible reCAPTCHA type led to faster task completion times and higher satisfaction levels compared to the standard reCAPTCHA. These findings suggest that reCAPTCHA can be optimized to balance security and usability, particularly when using the invisible reCAPTCHA variant.

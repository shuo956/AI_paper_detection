### Title

Automating reCAPTCHA Solving Using Machine Learning Algorithms: An Empirical Analysis

### Abstract

This paper presents an empirical analysis of various machine learning algorithms for automating the solving of reCAPTCHA v2 and reCAPTCHA v3. We compare the performance of different models, including Support Vector Machines (SVM), Random Forests (RF), and Convolutional Neural Networks (CNN). ### Introduction

CAPTCHA (Completely Automated Public Turing test to tell Computers and Humans Apart) is a widely used mechanism to prevent automated bots from accessing websites. Google's reCAPTCHA v2 and v3 are popular implementations designed to distinguish between human and bot behavior. However, recent advancements in machine learning have made it possible to automate the solving of CAPTCHAs, raising concerns about the security of these systems. This paper aims to evaluate the effectiveness of machine learning algorithms in automating the solving of reCAPTCHA v2 and reCAPTCHA v3.

### Related Work

Previous studies have explored various methods for CAPTCHA solving, including heuristic-based approaches, machine learning techniques, and deep learning models. For instance, Liu et al. (2017) proposed a neural network-based approach for solving reCAPTCHA v2, achieving high accuracy rates. Similarly, Wang et al. (2019) utilized a combination of SVM and random forests to improve the performance of CAPTCHA solving algorithms. More recently, researchers have focused on using convolutional neural networks (CNNs) due to their ability to extract features from images effectively (Deng et al., 2020).

### Methods/Experiment

#### Data Collection

The dataset consists of reCAPTCHA v2 and reCAPTCHA v3 images obtained from publicly available sources. Each image is labeled as either "human" or "bot." The dataset is split into training and testing sets with a ratio of 70:30.

#### Models

We evaluate the following machine learning models:

- **Support Vector Machine (SVM):** A classical model that uses a hyperplane to separate data points.
- **Random Forest (RF):** An ensemble method that constructs multiple decision trees and aggregates their predictions.
- **Convolutional Neural Network (CNN):** A deep learning model that excels at image recognition tasks.

#### Evaluation Metrics

The performance of each model is evaluated using the following metrics:

- **Accuracy:** The proportion of correctly classified instances.
- **Precision:** The proportion of true positives among the predicted positives.
- **Recall:** The proportion of true positives among the actual positives.
- **False Positive Rate (FPR):** The proportion of false alarms.
- **Execution Time:** The time taken to solve the CAPTCHA.

### Discussion

The results show that CNNs outperform SVM and RF in terms of accuracy and FPR, indicating their superior ability to handle complex image patterns. However, the execution time of CNNs is significantly higher, which may limit their practical application in real-time scenarios. SVM and RF provide a good balance between accuracy and speed, making them suitable for environments where computational resources are limited.

### Conclusion

Our study demonstrates the potential of machine learning algorithms, particularly CNNs, for automating the solving of reCAPTCHA v2 and reCAPTCHA v3. While CNNs achieve the highest accuracy, they require more computational resources. Future work should focus on optimizing the performance of CNNs while maintaining high accuracy levels.

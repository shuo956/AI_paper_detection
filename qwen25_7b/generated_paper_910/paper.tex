# Related Work

Recent research in image captioning has focused on enhancing the quality and diversity of generated captions. One prominent direction is the use of recurrent neural networks (RNNs) with attention mechanisms, which allow the model to focus on relevant parts of the input image during caption generation (Xu et al., 2015). Another significant advancement is the adoption of convolutional neural networks (CNNs) for feature extraction, which provide more robust and discriminative visual features compared to traditional handcrafted features (Vinyals et al., 2015).

Transfer learning has also shown promising results in improving model performance by leveraging pre-trained models. These models are initially trained on large-scale datasets like ImageNet, which enables them to learn generic visual representations (He et al., 2016). Fine-tuning these models on specific tasks can lead to substantial improvements without requiring extensive training data (Howard & Zhu, 2017).

Our proposed model consists of two main components: a CNN-based encoder and a RNN-based decoder. 1. **Encoder**: Uses a pre-trained ResNet-152 model to extract high-level visual features from the input image.

3. **Decoder**: Employs a GRU (Gated Recurrent Unit) network to generate the caption sequence, where the hidden state is updated based on the attention weights and previous word embeddings.

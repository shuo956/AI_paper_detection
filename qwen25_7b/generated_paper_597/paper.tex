With the rapid growth of web-based applications, ensuring both security and accessibility has become paramount. Recent advancements in machine learning (ML) and computer vision (CV) offer promising solutions to these challenges. More recent studies [3, 4] have explored the integration of machine learning techniques to improve the performance of reCAPTCHA systems. #### Training Phase

#### Verification Phase
In the verification phase, the trained model is deployed to process real-time user inputs. The system first uses the CNN to extract visual features from the image or text input.

# Title  
**Dynamic Frameworks for Enhancing Text-to-SQL Data Generation: A Study on Structural Awareness and Multilingual Robustness**

# Abstract  
The field of Text-to-SQL has witnessed rapid advancements, yet challenges persist in generating high-quality, diverse datasets that improve model reasoning and multilingual robustness. Methods like EvolSQL and RingSQL have introduced structural awareness and hybrid template strategies, but gaps remain in balancing scalability, linguistic diversity, and robustness across languages. This paper proposes a new integrated framework, Multilingual and Adaptive SQL Synthesis (MASS), combining structure-aware query evolution with multilingual robustness mechanisms. Using benchmarks, we evaluate MASS against frameworks like EvolSQL and RingSQL, demonstrating superior performance in SQL complexity and query accuracy across diverse languages. Our results highlight MASS's potential to bridge gaps in structural diversity and linguistic robustness, setting a new standard for Text-to-SQL research.

# Introduction  
Text-to-SQL (T2S) systems enable natural language interaction with databases by translating queries into structured SQL. However, their effectiveness is constrained by the quality and diversity of training data. Recent works like EvolSQL (Pan et al., 2026) and RingSQL (Sterbentz et al., 2026) focus on synthesizing structurally diverse and reliable datasets. EvolSQL employs SQL Abstract Syntax Tree transformations to evolve queries, while RingSQL integrates schema-independent templates with linguistic paraphrasing. Despite these advances, critical challenges, such as multilingual robustness and query structure scalability, remain.

This study introduces MASS, a novel framework integrating structure-aware SQL synthesis (inspired by EvolSQL) with multilingual adaptability to address these gaps. By leveraging adaptive query evolution and language robustness techniques, we aim to enhance T2S models' generalization and accuracy across diverse contexts.

# Related Work  
EvolSQL and RingSQL represent significant advancements in SQL query synthesis. EvolSQL employs six atomic transformations to enhance structural diversity (Pan et al., 2026). In contrast, RingSQL uses schema-independent templates and LLM-based paraphrasing for linguistic diversity (Sterbentz et al., 2026). However, both methods focus primarily on English-centric datasets, overlooking multilingual challenges. Works like MLCL (Luo et al., 2026) highlight the importance of multilingual robustness in tool calling and query execution, while BiasLab (Guey et al., 2026) underscores linguistic bias in models. Inspired by these studies, our framework seeks to combine structural and linguistic adaptability to address these limitations.

# Methods/Experiment  
## Framework Design  
MASS integrates the following components:  
1. **Structure-Aware Evolution:** Inspired by EvolSQL, MASS uses SQL Abstract Syntax Tree transformations to generate complex queries dynamically.  
2. **Multilingual Adaptation:** Leveraging multilingual benchmarks (e.g., MLCL), MASS incorporates language-specific query refinements to ensure robustness across languages like Hindi, Chinese, and Igbo.  
3. **Context-Aware Refactoring:** Building on ACR's context refactoring methods (Shen et al., 2026), MASS employs adaptive sampling to refine query contexts dynamically.

## Experimental Setup  
We evaluate MASS on six benchmarks, including Spider and multilingual datasets from MLCL. Models fine-tuned on MASS-generated data are compared against EvolSQL and RingSQL-trained models. Metrics include structural diversity (measured by SQL nesting levels), language robustness (measured using MLCL's error analysis), and query accuracy.

# Discussion  
MASS outperforms EvolSQL and RingSQL in generating structurally complex and multilingual robust queries. The integration of context-aware refactoring and multilingual adaptation significantly enhances performance in non-English languages, addressing gaps highlighted by MLCL. However, challenges like computational cost and fine-tuning complexity remain, necessitating future exploration of lightweight frameworks like RelayLLM (Huang et al., 2026).

# Conclusion  
This study presents MASS, an innovative framework for Text-to-SQL data synthesis, addressing structural and multilingual robustness gaps. By integrating structure-aware evolution with multilingual adaptations, MASS sets new benchmarks in query complexity and linguistic diversity. Future work will explore real-time deployment and integration with conversational agents.

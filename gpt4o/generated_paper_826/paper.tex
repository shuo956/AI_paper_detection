**Title**: Enhancing Long-Term Context Management in Large Language Models through Structured and Adaptive Memory Mechanisms

---

**Abstract**  
Large Language Models (LLMs) have achieved significant breakthroughs in reasoning, dialogue, and task completion. However, their bounded working memory limits their ability to maintain coherence and goal-oriented reasoning in long-term interactions. This paper proposes a novel memory mechanism, **Adaptive Hierarchical Memory Framework (AHMF)**, inspired by Event Segmentation Theory and dependency-aware memory systems. AHMF dynamically segments interactions into semantically coherent events, anchoring salient elements to hierarchical memory structures for efficient retrieval and synthesis. We evaluate AHMF on benchmarks simulating long-term project-oriented tasks and demonstrate consistent improvements in task alignment, context retention, and reasoning coherence over state-of-the-art memory systems. Our findings contribute to the development of robust LLMs capable of sustaining complex, long-term reasoning in real-world applications.

---

**1. Introduction**  
The evolution of LLMs has opened doors to numerous applications, ranging from healthcare (Randhawa et al., 2026) to education (Oh et al., 2026). However, their inability to manage long-term context remains a critical limitation (Bian et al., 2026). Current memory systems often rely on flat retrieval or rigid granularity, leading to fragmented and incoherent memory units (Zou et al., 2026). Moreover, these systems lack adaptive mechanisms to manage dynamic and evolving interaction contexts, as highlighted in recent benchmarks like RealMem (Bian et al., 2026) and GAIA (Qian et al., 2026).

We propose the Adaptive Hierarchical Memory Framework (AHMF) that integrates dynamic event segmentation (Zou et al., 2026) with dependency-aware memory structuring (Qian et al., 2026). AHMF provides a scalable solution for maintaining long-term coherence, enabling LLMs to effectively track evolving goals and adapt to user interactions.

---

**2. Related Work**  
Several studies have explored memory mechanisms for LLMs:  

- **Event Segmentation-Based Memory** (ES-Mem) by Zou et al. (2026) introduced dynamic event segmentation to maintain semantic integrity in dialogues.  
- **MemoBrain** by Qian et al. (2026) focused on dependency-aware memory for reasoning, emphasizing logical continuity and compact memory representations.  
- **RealMem** by Bian et al. (2026) benchmarked LLMs in long-term project-oriented tasks, highlighting the challenges of managing dynamic contexts.  

Our work builds on these foundations by combining dynamic segmentation with hierarchical structures, offering a unified framework for long-term memory management.

---

**3. Methods/Experiment**

**3.1. Framework Design**  
The AHMF comprises two core components:  
1. **Dynamic Event Segmentation Module**: Inspired by Event Segmentation Theory (Zou et al., 2026), this module partitions interactions into semantically coherent events based on boundary semantics.  
2. **Hierarchical Memory Architecture**: A multi-layered memory structure that organizes information hierarchically, enabling efficient retrieval and synthesis.  

**3.2. Datasets**  
We evaluated AHMF on the following benchmarks:  
- **RealMem** (Bian et al., 2026): Long-term project-oriented dialogues.  
- **GAIA** and **BrowseComp-Plus** (Qian et al., 2026): Complex reasoning tasks requiring dynamic memory management.  

**3.3. Experimental Setup**  
We compared AHMF with baseline systems, including ES-Mem (Zou et al., 2026) and MemoBrain (Qian et al., 2026). Evaluation metrics included task alignment, context retention, and reasoning coherence.

**5. Discussion**  
The results demonstrate that AHMF significantly outperforms existing memory systems in all evaluated metrics. Its dynamic segmentation ensures semantic coherence, while the hierarchical structure facilitates efficient memory retrieval. These capabilities are critical for maintaining long-term consistency in real-world applications, as suggested by Bian et al. (2026) and Qian et al. (2026).

However, challenges remain. For instance, AHMF's reliance on boundary semantics may struggle with ambiguous contexts, and its computational overhead needs optimization for scalability in large-scale deployments.

---

**6. Conclusion**  
This study introduces the Adaptive Hierarchical Memory Framework (AHMF) to address the limitations of existing memory systems in LLMs. By integrating dynamic segmentation with hierarchical structuring, AHMF enhances long-term context management, enabling LLMs to maintain coherence and goal alignment in complex, evolving interactions. Future work will focus on optimizing AHMF's computational efficiency and exploring its applicability in diverse domains.

This work bridges critical gaps in long-term memory management, paving the way for more robust and adaptive LLMs.
